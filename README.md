## Summary

Weatherology uses dark sky api to display the current weather and forecast.

![alt tag](https://github.com/ab218/weatherology/blob/master/src/Assets/weather.gif)

## Installation

- Clone repo

- Create .env in project root (see env.example)

```
yarn start
```
